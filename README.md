Modern full-stack platform for Erasmus. Responsive dashboard, fast API, JWT+RBAC authentication and PostgreSQL.

---

## 🎯 Project Objective

- **Centralize** services for Erasmus students in Třeboň
- **Modern dashboard** with Next.js + Tailwind CSS
- **Robust API** FastAPI with automatic documentation
- **Secure authentication** JWT + RBAC (Admin, Coordinator, Student, Teacher)

---

## 🛠️ Technologies

**Frontend**: Next.js 14 + React 18 + Tailwind CSS  
**Backend**: FastAPI + Python 3.12 + Pydantic  
**Database**: PostgreSQL 16 + SQLAlchemy  
**Auth**: JWT tokens + RBAC (4 roles)
